    Implementeer de gegeven interface van de klasse PRquadtree.

    Opmerking: Als knoopptr een Knoop* is die wijst naar een blad, dan kan je aan de x-coördinaat van het punt met static_cast<Blad*>(knoopptr)->x.

    Een tip die je kan helpen: 

    class Gebied{
public:
    Gebied(int maxc):xl(-maxc),xr(maxc),yl(-maxc),yr(maxc),xc(0),yc(0){};
    void maakdeelgebied(int x,int y){
            if (x<xc)
                xr=xc;
            else
                xl=xc;
            if (y<yc)
                yr=yc;
            else
                yl=yc;
            xc=(xl+xr)/2;
            yc=(yl+yr)/2;
    };
    int geefKwadrant(int x, int y) const{
        return (x<xc? WEST : OOST) + (y<yc? NOORD: ZUID);
    }
private:
    int xl,xr,yl,yr,xc,yc;//geven grenzen gebied; l,r,c: links, rechts, centraal.
}
    Bijkomende opgave:

    Reverse software engineering is het afleiden van programmeerbeslissingen uit de resultaten. De bijkomende opgave is een voorbeeld hiervan: bepaal welke de waarden van NOORD, OOST, ZUID en WEST zijn aan de hand van de tekening hiernaast. Controleer je antwoord door te zien of jouw code dezelfde boom oplevert voor dit geval.
