  • Ternaire bomen                         29 november 2013

  • (Noot: dit is een korte oefening die geen geheel labo inneemt). Hiernaast zie je een ternaire zoekboom. De strings zijn afgesloten met het \#-karakter, dat kleiner is dan alle letters. Ga ne (1) welke strings zijn opgeslagen in deze ternaire boom en (2) in welke volgorde ze zijn toegevoegd.

  • Gegeven de volgende datastructuren voor een ternaire zoekboom: 
    class TernaireBoom{
    TernaireBoom(){k=0;};
protected:
    TKnoop* k;
    const static char afsluiter;
};
const char TernaireBoom::afsluiter=char(127);

class Tknoop{
friend class TernaireBoom;
protected:
    char c;
    TernaireBoom kind[3];
};
    Implementeer TernaireBoom::voegtoe(const string&). Je mag veronderstellen dat TernaireBoom::afsluiter niet in een string kan voorkomen. Merk op dat TernaireBoom::afsluiter groter is dan elk ander karakter.
